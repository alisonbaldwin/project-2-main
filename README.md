# Iowa Water Quality Monitoring Dashboard

## Overview

The Iowa Water Quality Monitoring Dashboard is an interactive web-based application designed to visualize and analyze water quality data across the state of Iowa. This project transforms complex environmental datasets into an intuitive and user-friendly interface, allowing users to explore, filter, and compare water conditions in real time.

Instead of relying on raw spreadsheets, users can interact with a dynamic map and dashboard to quickly understand key water quality metrics and geographic patterns.

---

## Key Features

- Interactive map with clickable monitoring stations  
- Real-time filtering by water quality conditions  
- Search functionality to locate specific stations  
- Dynamic statistics and distribution summaries  
- Visualization of key metrics such as nitrate levels, pH, and dissolved oxygen  
- Color-coded markers for quick interpretation of water conditions  

---

## Technologies Used

- **HTML** – structure of the application  
- **CSS** – layout, styling, and responsive design  
- **JavaScript** – application logic and interactivity  
- **Leaflet.js** – map visualization and geographic rendering  
- **PapaParse** – parsing CSV data into usable formats  
- **CSV & GeoJSON** – data storage for water quality and location information
